CRM для сотрудников и отделов

Ссылка на видео: https://youtu.be/Q4JTtyBTnBo

Микросервис на базе Spring Boot, предназначенный для автоматизации учета сотрудников внутри организации. Можно управлять структурой компании, связывая сотрудников с конкретными отделами и должностями.

Можно добавлять новые отделы, добавлять и удалять должности, а так же добавлять сотрудников, менять их данные и удалять сотрудников
